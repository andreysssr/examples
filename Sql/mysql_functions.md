<a name="arithmetic_operators"><h2>Арифметические операторы</h2></a>

```
%, MOD	Оператор по модулю `MOD(N,M)`, `N % M`, `N MOD M`
*	Оператор умножения
+	Оператор сложения
-	Меньше оператора
-	Изменить знак аргумента
/	Оператор деления
DIV	Целочисленное деление `SELECT 5 DIV 2`
```
